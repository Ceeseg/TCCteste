Senhas seguras











Criar uma senha segura é vital para proteger suas informações pessoais e financeiras em comércio eletrônico. Aqui estão algumas dicas sobre métricas de senha, para a criação de uma senha segura:


1. Alguns sites durante a criação de senha pedem a utilização de uma combinação de caracteres que são a base para a criação de uma senha segura, onde contém:

- Letras maiúsculas e minúsculas, uma ideia seria alternar entre elas;
- Números em lugares não convencionais, não apenas no final;
- Utilização de símbolos e caracteres especiais como ! @ # $ % ^ & *.

2. Comprimento da senha:

Uma senha para ser considerada segura deve ter no mínimo 12 caracteres. Quanto mais longa, mais eficaz.

3. Evite informações pessoais:

Não utilize datas de nascimento, nomes de pets, ou qualquer informação facilmente encontrável em redes sociais.

4. Não utilize palavras completas:

Palavras inteiras tornam as senhas mais vulneráveis a ataques.
Exemplo: Joãozinho123

5. Utilize de estratégias para criação de uma senha que não seja facilmente esquecida:

Utilize de coisas no seu cotidiano, como uma música ou uma frase, porém transformando em uma senha, pode ser usando a primeira letra de cada palavra de uma frase que faça sentido para você, colocando com letras maiusculas, minúsculas e caracteres especiais;
Exemplo: "Eu amo acordar as 7 da manhã!"poderia se tornar "Ea2@7dM!".

6. Atualização regular:

Troque suas senhas a cada 2 ou 3 meses para maior segurança.


Métodos para gerenciar senhas seguras:


1. Use um gerenciador de senhas:

Ferramentas como LastPass, 1Password, ou KeePass podem gerar e armazenar senhas complexas para você.

2. Autenticação de dois fatores (2FA):

Habilitar o 2FA, vai trazer uma maior segurança para sua senha caso ela seja descoberta, pois para o acesso seria necessario o segundo nível de segurança.

3. Varie as senhas para diferentes sites:

Nunca reutilize a mesma senha. Caso em algum momento sua senha for descoberta, todas as suas contas estarão em risco.


Testando a força da senha:


- Use ferramentas online para testar a força da senha, mas tenha cuidado para não usar suas senhas reais, apenas variações delas;
- Quanto mais tempo demorar para uma senha ser "quebrada" pela ferramenta, mais forte ela é considerada.
- Para testar a força da senha acesse: https://password.kaspersky.com/pt/
